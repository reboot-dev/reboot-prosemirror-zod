# Reboot + ProseMirror (w/ Zod)

### Requirements
- macOS or Linux
- Node.js >= 20.10.0
- Docker

Install packages (you may need to run `corepack enable` in order to run `yarn`):

```console
yarn
```

In one terminal run:

```console
yarn run dev:backend
```

And in another terminal:

```console
yarn run dev:web
```

To run the tests:

```console
yarn run test:backend
```
